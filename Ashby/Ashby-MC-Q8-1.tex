%Solution------------------------------------------------
\begin{solution}

On option is to just realize that if the number in the denominator is 1, you invert the power value in the denominator and you get your value. This is using the equation: $\text{slope} = \frac{1}{a}b$ if the index is $\frac{\alpha^{\frac{1}{3}}}{b}$ . $a = \frac{1}{3}$ and $b = 1$, the slope is three. This corresponds to slope \#1.

Alternatively --- and students should be able to do this --- we can do the following:
%
\begin{align*}
	M_{\text{i}} &= \frac{\alpha^{1/3}}{\sigma_y}\\
	\text{log}(M_{\text{i}}) &= \frac{1}{3} \text{log}(\alpha)-\text{log}(\sigma_y)\\
	\frac{1}{3} \text{log}(\alpha) &= \text{log}(\sigma_y)+\text{log}(M_{\text{i}})\\
	\text{log}(\alpha) &= 3\text{log}(\sigma_y)+\text{log}(M_{\text{i}})
\end{align*}
%
So the correct line is the one with slope 3. This is line 1.

\end{solution}
